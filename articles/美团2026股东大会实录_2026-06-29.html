美团2026股东大会实录：王兴的坦诚与反思

2026-06-29 ｜ 持仓公司研究 · 美团 · 股东大会

📝 核心摘要

2026年6月26日，美团召开股东周年大会。王兴罕见低头认错，承认过去两年因外卖补贴大战错失出海、AI两大战略窗口期。2025年美团从盈利358亿转为净亏损234亿，核心本地商业从盈利524亿转为亏损69亿。王兴明确：股价被严重低估，将重启回购；手握650亿可变现投资资产；全面转向理性竞争+AI深耕。对投资者的核心启示：短期看行业竞争是否真回归理性，中期看UE转正拐点，长期看AI能否构建新护城河。

📉 一、股价现状与王兴表态

股价跌幅

  • 2021年巅峰：460港元/股，市值近3万亿港元
  • 2026年6月26日：64.25港元，市值约3967亿港元
  • 距发行价69港元仅一步之遥，较巅峰跌幅超85%
  • 2026年内跌幅：超37%

王兴罕见认错

王兴原文："过去几年，公司股价表现不理想，我责任重大。"

  • 没有推诿宏观环境，没有甩锅行业周期
  • 承认过去两年"无效浪费"——陷于外卖补贴大战，白白消耗
  • 澄清：自美团创立以来一股未卖，2021年捐出10%股权至公益基金会（不可撤销）
💥 二、2025年惨烈财报：两年补贴大战的代价

关键数据

王兴直言：外卖行业利润率仅约3%，这场补贴大战从一开始就没有赢家。美团守住60%+外卖GTV份额，但近600亿利润凭空蒸发。

🚨 三、王兴亲口承认：两大战略失误

失误一：出海布局严重滞后，错失窗口期

  • 最佳窗口期：2019-2023年，海外本地生活/外卖赛道爆发拐点
  • 彼时美团全身心扑在国内补贴战，无暇顾及海外，直接搁置全球化
  • 等到2024年美团"骤然醒悟"，海外赛道格局已定，先发优势彻底归零
  • 对手早已超车：字节深耕全球电商、拼多多狂奔Temu、携程站稳海外文旅

当前出海进展（迟到但启动）：

  • 香港Keeta：1年登顶单量第一，已率先实现UE转正
  • 中东市场：稳步落地，UE改善速度快于香港
  • 巴西：官宣五年10亿美元重注，以圣保罗为试点
  • 策略：先实现单市场盈利再复制，极度保守稳健

失误二：美团优选五年探索，终究未能跑通

  • 社区团购模式对价格极度敏感，非标品极易陷入低价竞争
  • 美团不愿以低价损耗商品品质，原有模式难以持续盈利
  • 五年投入数百亿，最终黯然退场，成为沉没成本
  • 转型方向：快乐猴社区店（700-800㎡，少SKU、高自有品牌占比、极致性价比）

王兴反思：不能只做卖家竞价，必须深入供应链，做极致性价比。

失误三（隐性）：AI踏空，有底牌却未发力

  • 美团其实优势得天独厚：百万商家数据、海量交易场景、成熟履约体系
  • 但过去两年，公司所有资源全部倾斜外卖补贴，AI战略被边缘化
  • 等到字节、腾讯、阿里、DeepSeek群雄割据，AI壁垒已然成型
  • 王兴在股东会上坦言："过去两年，属于美团的无用浪费"
🃏 四、美团的四张自救底牌

底牌1：主动停战，终结无效内卷

  • 2026年3月，监管定调"外卖大战该结束了"
  • 王兴在股东大会再度公开呼吁行业理性竞争
  • 目前三家（美团、淘宝闪购、京东）一小时内齐刷刷表态"坚决拥护"
  • 6月17日，国家市场监管总局发布《外卖平台补贴行为规范十条（征求意见稿）》

行业风向已逆转：美团缩减补贴、深耕即时零售；阿里放弃外卖内卷、All in AI；京东直言不争短期朝夕。

底牌2：真金白银回购，创始人兜底

  • CFO陈少晖表态："公司价值被严重低估"，持续推进股票回购
  • 2024年1月至2025年5月，累计回购近36亿美元，覆盖约4%股本
  • 常规年度回购底线为1%股本（去年因大额补贴暂停，当前持有董事会回购额度）
  • 王兴重磅兜底：创业至今一股未卖、未来也绝不减持

底牌3：手握650亿隐形资产，随时可套现回血

  • 截至2026年3月，美团在AI、硬科技、机器人赛道的投资组合公允价值接近650亿元
  • 核心持仓：理想汽车约12%、智谱AI约3.9%、宇树科技约7.6%、古茗、蜜雪冰城等
  • 其中宇树科技+智谱AI两笔核心投资，对应市值就近350亿
  • 解禁后将结合现金流、市场估值分批退出变现，回流资金用于经营或回购

这是美团最被市场低估的隐藏底牌。

底牌4：业绩持续减亏，基本面触底反弹

  • 2026年Q1：经营亏损环比大减96亿，核心本地商业亏损大幅收窄
  • 摩根士丹利、瑞银等多家投行维持"买入"评级
  • 预判2026年Q2核心业务有望盈亏平衡，常态化利润有望突破400亿
🤖 五、未来战略：AI是最后的翻盘机会

AI战略全景

  • 组织：成立一级战略部门"AI Transformation"，与外卖核心业务平行，举全公司之力攻坚
  • 技术：自研万亿参数大模型LongCat 2.0-Preview落地；联手腾讯元宝AI补齐通用能力短板
  • C端产品："小团"升级为核心流量入口，支持语音交互，抢占用户服务入口
  • B端产品：智能掌柜、AI小管家、袋鼠管家全面落地，服务超百万商家

美团AI的差异化路线

不同于阿里主打区域运营、京东主打用户体验，美团走出了"产业AI落地实体经济"的差异化路线：

  • 不玩概念、不炒热度
  • 专注服务线下中小商家
  • 用技术沉淀（而非补贴）构筑壁垒

王兴判断："未来本地生活的竞争，不再是补贴大战，而是技术与效率的战争"

三层护城河重构

层级内容
✅ 物理壁垒数百万骑手+无人配送、无人机成熟履约网络
✅ 数据壁垒百万级商家经营数据、用户消费数据沉淀
✅ 未来壁垒AI Agent全域赋能，抢占下一代服务入口
🛒 六、即时零售双业务：小象超市 + 快乐猴

小象超市（前置仓模式升级）

  • 线上前置仓广覆盖 + 线下实体店落地（北京、宁波已开店）
  • 线下门店建立商品品质信任，线上承接复购，实现优势互补
  • 在品质管控上，对标山姆模式，深耕供应链，摆脱单纯价格内卷
  • 参照美国沃尔玛单年销售额数千亿美元，国内生鲜零售市场空间广阔

快乐猴（社区店模式，承接优选转型）

  • 700-800㎡社区店，少SKU、高自有品牌占比，极致性价比
  • 与小象形成错位竞争
  • 底层逻辑：不做卖家竞价，深入供应链，做极致性价比

王兴反思优选败局后的核心启发："不能只做卖家竞价，必须深入供应链"

🏨 七、到店酒旅：最优质的现金牛

与抖音的差异化竞争

  • 抖音：依托内容流量做一次性低价促销
  • 美团：一站式本地生活交易平台，打通会员、商家经营、售后保障
  • 美团优势：复购与长期经营，而非一次性流量变现

经营数据

  • 利润率：常年保持20%-30%，是稳定现金牛
  • 策略：不再为冲份额无效投放，严格考核投入产出，保持合理利润率
  • 协同优势：外卖高频LBS流量反哺到店业务，形成其他平台不具备的综合壁垒

2026年隐忧：抖音趁美团被外卖战拖累，大幅加码到店补贴；美团内部资源被迫从到店抽走投外卖。到店经营利润率从2022年46% → 2025年Q4的25%，2026年预计历史首次年度利润同比下降。

🍔 八、外卖竞争现状：份额稳固但盈利修复进行中

市场份额

  • 美团整体GTV份额：约60%+（守住基本盘）
  • 30元以上高价值优质订单份额稳定在70%以上（高端市场护城河极深）
  • 友商（阿里淘宝闪购）单位经济模型持续落后，规模、网络效应、用户复购留存构筑的护城河依然稳固

UE（单均利润）修复进度

  • 2026年Q1：UE大约接近盈亏平衡点
  • 即时零售部分UE表现比纯外卖更好
  • 摩根士丹利预判：2026年Q2核心业务有望盈亏平衡

但UE优势不是绝对壁垒——竞对持续投入+优化，3年左右大概率可以追上。美团幸运的是，竞对投入战场的火力大概率在减弱。

💡 九、对投资者的主要启发

启发1：管理层坦诚度极高，这是巴菲特最看重的品质

王兴在股价跌去85%后，能说出"责任在我"，并在股东大会上详细复盘两大战略失误。这种坦诚在A股/港股上市公司中极为罕见。长期持有最重要的不是管理层聪明，而是管理层诚实。

启发2：美团护城河是"成本护城河"，不是"用户粘性护城河"

2025年补贴战证明：竞争对手花足够的钱可以切走市场份额，美团被迫跟进，定价权在对手手里。这与茅台（品牌定价权）、腾讯（网络效应定价权）有本质区别。美团的投资逻辑是"效率优势"，不是"不可替代性"。

启发3：650亿投资组合是隐藏价值，但是一次性收益

智谱AI、宇树科技、理想汽车等资产价值超650亿，这是市场严重低估的。但这些收益只是一次性，只能短期美化财报，无法改变核心业务的长期盈利能力。不能因为"有资产可卖"就盲目乐观。

启发4：到店业务已被战火波及，这是最大的隐忧

2026年美团到店业务将出现历史首次年度利润同比下降（疫情期间2020/2022都未发生过）。原因：抖音趁美团被外卖战拖累，大幅加码到店补贴。唯一拐点：外卖补贴退坡 → 美团将资源重新投入到店 → 抖音感知"再投也拿不到更多份额" → 抖音回归ROI导向 → 到店竞争缓和。

⚠️ 风险提示：三个不确定性

1. 阿里/京东降补信号是否真实？阿里明确"三年不计亏损加码淘宝闪购"，这意味着即使美团不想战，对手也可能主动挑起战争。需持续跟踪阿里/京东的补贴力度变化。

2. AI投入能否转化为利润？美团AI战略起步晚，短期（1-2年）很难形成颠覆性优势。AI变现能力仍需长期验证。不能因为"王兴说要做AI"就盲目乐观。

3. 出海能否真正贡献利润？巴西市场5年10亿美元投入，短期（3年内）很难贡献实质性利润。出海是长期故事，不是短期催化剂。

🔭 十、后续需要持续跟踪的方向

短期跟踪（2026年Q2-Q3）

  • 🔍 Q2外卖UE是否如期转正？（最重要验证点，决定盈利拐点是否到来）
  • 🔍 阿里/京东降补信号（淘宝闪购月度GTV、补贴力度变化）
  • 🔍 到店经营利润率是否企稳？（是否止住2025年Q4以来跌势）
  • 🔍 抖音本地生活月度亏损是否收窄？（决定抖音是否继续加码到店补贴）

中期跟踪（2026年Q4 - 2027年）

  • 🔍 小象超市门店扩张进度 + UE数据（是否能成为第二增长曲线）
  • 🔍 快乐猴门店数量 + 单店模型（优选转型是否真能跑通）
  • 🔍 Keeta中东 + 巴西市场份额 + UE（出海能否真正盈利）
  • 🔍 650亿投资组合退出进度（智谱AI、宇树科技解禁后是否真的变现）

长期跟踪（2027年及以后）

  • 🔍 "小团"AI助手用户留存率（是否真的成为下一代服务入口）
  • 🔍 B端AI工具商家付费率（AI是否真的帮商家降本增效，并让美团分成）
  • 🔍 无人配送商业化进度（无人机、无人车是否真的降低履约成本）
