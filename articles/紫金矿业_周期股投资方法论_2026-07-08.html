周期股该怎么投 — 以紫金矿业为例

低 PE 时抛出，还是长期持有看经营与储量？这是一篇方法论笔记

一句话结论：你直觉里的"低 PE、资源价高时抛出"——那是教科书级的纯周期股正确做法。但紫金不是纯周期股，它是"周期 + 成长"的混合体，所以答案不是二选一，而是一个带条件的第三条路。而对你（自认不擅长周期股）来说，最务实的是"质量 / 归一化估值法 + 小仓位"，而不是去赌自己判断不了的金铜拐点。

一、纯周期股的铁律：低 PE 往往是卖点，不是买点

这是彼得·林奇在《战胜华尔街》里讲透的：周期股要在高 PE（利润谷底）时买、低 PE（利润顶峰）时卖。因为周期股的利润大幅摆动，PE 是"利润的倒数镜像"——

  • 行业最惨时，利润微薄甚至亏损 → PE 奇高甚至为负 → 其实是真便宜（资产、成本优势还在）；
  • 行业最火时，利润爆表 → PE 低得诱人 → 其实是陷阱（利润马上要均值回归）。

"低市盈率、金铜价高企之时及时抛出，等周期低点再进"——完全正确，这是对纯周期股的标准纪律。问题在于这条纪律极难执行：低的时候人恐惧不敢买，高的时候人贪婪舍不得卖，"每一次都觉得自己这次不一样"。雪球有篇文章说得好：这需要的纪律性"跟减肥一样——道理都懂，执行不了"。这恰恰是你说"不擅长周期股"时最该敬畏的一点。

二、但紫金不是纯周期股——它有"量"的成长性

另一派（雪球"别再叫它周期股"）的论点是：紫金过去十年净利润从 18 亿 → 近 800 亿（机构预期），40 倍增长远不止靠涨价，核心是"量"的狂飙——

指标2016 年2026 年规划
金权益产能42.6 吨105 吨
铜权益产能15.5 万吨120 万吨
锂无12 万吨（新增）

也就是说，即使金铜价格十年不动，光靠产能释放，利润也该翻几倍。它像一家"产品线在扩建的工厂"，涨价只是锦上添花。所以这一派主张：别被短期金属价格波动吓跑，看未来三五年实实在在的产量和利润增长，长期持有。

三、芒格 / 巴菲特怎么看——给你的框架定调

你的投资框架是巴芒，所以直接用他们的尺子量：

  • 芒格原话"即使在自然资源行业，连我们自己的投资回报也不好。" 巴菲特买康菲石油（2008 高位）油价腰斩，认亏 40% 割肉；西方石油熬到 2026 才靠优先股股息拉正。大宗商品企业是大师都栽过跟头的地方。
  • 但他们不排斥"低成本生产商"：芒格专门夸过浦项钢铁（低成本 + 差异化），巴菲特买过陕煤（成本最低一档 + 区位）。关键判断是那篇雪球文章提的三问：
    1. 产品有没有差异化（客户是不是只认价格）？
    2. 成本优势能否长期持续（靠运气还是靠本事）？
    3. 利润是不是真金白银的自由现金流？

紫金对这三问的回答：①产品无差异化（硬伤，纯商品）；②成本优势能持续（卡莫阿 C1 成本 1.4 美元/磅，全球前 quartile，靠本事；资源禀赋不可复制）；③自由现金流强劲（2026Q1 经营现金流 278 亿）。 → 所以它是"半个合格"的稀缺标的：成本护城河够硬，但商品属性去不掉。

四、给你的实操方案（不擅长周期股者的生存法则）
（1）先决定"你信不信"——这是持有与否的前提
  • 你信不信：长期铜需求（能源转型 / AI 数据中心 / 电网）和黄金（央行购金 / 债务货币化）的需求是真实的、多年的？
  • 你信不信：紫金的低成本 + 产能释放能兑现？
  • 两个都信 → 可以小仓位长期持有；有一个不信 → 干脆别碰（能力圈边缘的事，做了也难赚钱）。
（2）估值锚定"归一化利润"，别锚定峰值

这是对你最重要的纪律。当前 8.7 倍 forward PE，是用 ~820 亿峰值利润算的，是"假便宜"。正确做法：

  • 用 mid-cycle（中庸周期）利润估值，比如假设金铜回到长期均价后的年利润（可能 500-600 亿量级）；
  • 在这个口径下算 PE，再决定贵贱。当股票因为金价暴跌而 PE 看起来很高时，那往往才是真买点；当 PE 低得诱人时，反而要警惕。把你现在的直觉整个反向用。
（3）仓位要小，且把它当"卫星"不是"核心"

你 100 万计划里紫金 12%、核心仓是腾讯茅台——这个结构本身就是对的。周期股波动远大于消费 / 互联网核心仓，仓位上限就该低于核心仓。你现在的配置已经体现了这点。

（4）盯着"不变的基本面"，让价格当噪声

你原话里提到的恰恰是最该看的：经营管理能力、矿产资源储备、加工成本控制。补充几个具体盯盘指标：

  • 储量寿命 / 资源禀赋是否还在增长（逆周期并购是否持续低价拿矿）；
  • C1 成本排名是否保持行业前 quartile；
  • 资产负债率（已降到 51%，是否失控）；
  • 管理层并购纪律——最危险的信号是"在周期顶部高价收购"，这会在下一轮低谷引爆资产负债表；
  • 这四条稳，紫金就还是那个紫金；这四条坏，才是真正的卖点（不是金价跌就卖）。
（5）下跌对你可能是机会，不是警报——前提是你有余钱和 nerves

你看到 44 → 25 的回调，对纯交易者是要逃命；对"质量 + 小仓位"持有者，反而是用 normalized 估值看是否加仓的窗口。但必须用"时间杠杆 + 认知杠杆"，绝不用融资杠杆（雪球那篇建仓文也强调）。

五、落到当前紫金的具体判断
  • 现在 ~8.7 倍 forward PE，处在"低 PE + 金铜高位"的经典黄灯区；
  • 绿灯信号：量的释放（巨龙二期、锂 12 万吨）、低成本、真金白银回购；
  • 黄灯信号：金价较高点 -29%、Q2 必然弱于一季度、70% 是高峰值对比；
  • 所以我的判断是：不是"明天必须卖"的红线，也绝不是"PE 才 8 倍赶紧满仓"的绿灯。 若你信长期逻辑，12% 的小仓位拿着、价格大跌时分批加点、平时不动，是匹配你"不擅长周期"特质的最稳打法；若你发现自己其实无法忍受单日 ±9% 的波动、总想猜顶抄底，那减到更低甚至清掉，把火力集中到茅台腾讯这种你更懂的生意，反而是更理性的选择。
