泡泡玛特二级市场专题

折价率是指标里更重要还是次要的？——兼谈去哪查准的二手价格
📅 2026-07-12 🏷️ 泡泡玛特（09992.HK）
#二级市场 #折价率 #指标权重 #需求前瞻 #讨论笔记
折价率是"更重要"还是"次要"？——以及，去哪查到准的二手价格走势

二级市场折价率，是泡泡玛特最特殊、也最容易被忽视的一个指标。本文分三部分：先讲清楚二级市场到底是什么、为什么会存在、价格为何涨涨跌跌；再回答核心问题——在衡量泡泡玛特时，折价率相比销售收入、利润、新IP成功率，到底是更重要还是次要；最后给出一份实操指引：到底去哪能查到比较准的二手价格走势。

一、二级市场是什么，为什么会存在

二级市场是相对于"一手市场"而言的。泡泡玛特官方以零售价把盲盒卖给消费者，这是第一道交易（一级市场），价格由官方定。但泡泡玛特的特别之处在于：很多人买来不只是（或根本不是）为了自己玩——有人为了凑齐系列、有人赌"隐藏款"、黄牛为了转卖赚钱。于是这些买来的人，会把手里多余的、或抽到的稀缺款，拿到闲鱼、得物、千岛（原葩趣）、线下潮玩店再卖出一道，这就是二级市场（二手转卖市场）。

关键差别：一级市场价格由官方定（零售价），二级市场价格由供需自由博弈——谁出价高谁拿走。泡泡玛特的罕见之处，是把"盲盒稀缺机制"做成了一个面向大众、高频、价格高度波动的二级市场，所以它才被当作一个观察指标。

顺带纠正一个直觉误区：不是"其他品牌没有二级市场"，而是大多没有"成规模、可量化、能反映情绪的二级市场"。球鞋（AJ）、集换卡（宝可梦）、手表、奢侈品包、甚至乐高绝版 set 都有二级市场。泡泡玛特把这套机制做成了大众级、可追踪的情绪温度计。

它为什么会存在？四个底层原因：① 盲盒的"信息差"机制——付钱时不知抽到哪款，隐藏款概率极低（常见 1/144），为了集齐或赌隐藏，人们成盒成盒地买，必然产生大量重复款流进二级市场；② 稀缺性被设计进产品——隐藏款、限定款、地区款、联名款人为制造稀缺；③ 社交货币属性——当 Labubu 成为"拎出门就被问"的社交符号，拥有一个=身份表达，愿意为"拥有"付溢价；④ 投机/收藏文化——黄牛和收藏者把潮玩当"类资产"，低买高卖，和炒鞋、炒卡逻辑一样。

二、价格为何涨涨跌跌

一个 IP 在二级市场的价格，本质是稀缺叙事 × 情绪热度 × 资金博弈三者的乘积，通常走"炒作—崩落—折价"的生命周期：

  • 首发→拉升：新品缺货 + 隐藏款极低概率，黄牛和玩家涌入抢购，价格被炒到零售价的 2~4 倍。多个产品系列的溢价高点出现在 2025年7月前后，这也是泡泡玛特股价的高点——二级溢价和股价高度同步，这正是它成为"前瞻指标"的原因。
  • 见顶→回落：泡泡玛特一旦补货、提产，稀缺被打破；同时黄牛在高位集中抛售套现，价格开始塌。
  • 退潮→折价：当"这东西以后会升值"的信仰破裂，持有者不再惜售、只求回血，价格跌破零售价。今年（2026）初开始折价，Labubu 样本 SKU 折价目前普遍在 20%~45% 区间。
真实数据印证（来源：财联社、新浪财经、界面新闻、央广网，引用得物/千岛/闲鱼 APP 数据，2025–2026）：
Labubu 3.0「前方高能」单个盲盒（得物）峰值 639.3 元（溢价 5.5 倍）→ 年末 116.1 元，跌 81.8%
Labubu 3.0 隐藏款「本我」（得物/新浪）峰值 4522~4733 元 → 523~570 元，跌 87~89%
Labubu 4.0 迷你版整盒（新浪）3200 元 → 1077 元，跌约 66%
Labubu 冰箱（未发货阶段"纸面"峰值）开售 3 小时 20999 元 → 7300 元，蒸发超 60%
注意：折价 ≠ 卖不出去。一级市场可能还在正常出货，但二级市场"折价"说明——买到手的人，已经不相信它未来能保值/增值了。这恰恰动摇的是 IP 的"收藏价值"和"社交货币"属性，而这正是泡泡玛特区别于普通玩具公司、撑起溢价估值的核心资产。

三、折价率：更重要，还是次要？

直接结论：没有谁"绝对更重要"，但针对不同问题，权重完全不同。而针对我们持有泡泡玛特时的核心纠结——"130亿是不是可持续的利润底"——二级市场折价率，是这套指标里权重最高的那个。原因不是它最准，而是它说话最早，且逼问的是一个利润表回答不了的问题。

其他三个指标（月度销售、利润、北美单店）有一个共同特征：它们都是一级市场的"结果数据"——公司把货卖出去、确认收入后才产生。折价率不一样，它是货已经到消费者手里之后，由"持有者愿不愿意为它付溢价"投票出来的价格发现。它剥离了"官方还在铺货、经销商还在囤货"的噪音，直接逼问那个最尖锐的问题：终端拿到手的人，还信不信这件东西未来能保值、有社交货币属性？

和另外三个指标逐一对比

你问的是哪个更重要理由
该警惕吗？方向对不对？折价率它最早说话，且直击"需求信仰"这个核心；利润表要滞后 6–12 个月才确认
这家公司值多少钱？利润 / 单店它们是现金流的事实，最权威；折价率再吵也给不出估值锚

  • 折价率 vs 月度销售 / 单店：月度销售告诉你"官方这月卖了多少"，但被铺货节奏、经销商库存、促销节点平滑。单店销售（流量×转化×客单）好，说明公司当下赚钱，但滞后于需求拐点。定位：销售/单店是"确认"，折价率是"预警"。
  • 折价率 vs 利润：利润最权威（估值的现金流锚），但也最滞后（季度确认 + 会计平滑）。折价率预警的拐点，利润要 6–12 个月后才"承认"。定位：利润用来"定估值"，折价率用来"定方向"。这正好接上我们存过的"安全边际"那篇——巴菲特看的不是当期利润，是"可持续的利润底"，而折价率恰恰是验证那个"底"还在不在的最早信号。
  • 折价率 vs 新 IP 开发成功率：新 IP 成功率是结构性慢变量，决定天花板。它和折价率互为因果：整体折价扩大 → 潮玩情绪退潮 → 新 IP 接力更难。看新 IP 成功率，得先看折价率给的"情绪水位"。
对泡泡玛特这种强情绪驱动、且去年利润含 Labubu 峰值的公司，你最该怕的不是"利润短期波动"，而是"130亿的利润底悄悄被重定价"。在这个最致命的问题上，折价率的权重高于利润——因为等你从利润表里看到恶化，股价早反映了。

但必须强调折价率的缺陷，所以它"必要但不充分"：受黄牛操纵、单一 IP 波动大、闲鱼/得物样本有偏差。三者要共振才可信：折价率持续扩大 + 单店坪效连降 + 新 IP 动销疲软，同时出现才是真正的危险信号；单看折价率扩大，可能只是某个 IP 的自然退潮。

四、去哪查比较准的二手价格走势（实操指引）

这是最实用的一节。基于 2025–2026 年各权威媒体（财联社、新浪财经、界面新闻、央广网）实际引用的数据源，当前比较可靠的渠道如下：

① 得物（毒）APP —— 媒体引用最多、最常被采用

几乎所有关于泡泡玛特二手价的公开报道，引用的都是"得物 APP 数据显示"。它胜在：有成交均价的历史曲线（不止当前挂牌）、潮玩/鞋服品类成熟、价格体系相对透明。看单个系列（如"前方高能""Labubu 3.0"）能看到从发售到回落的完整走势。对投资者而言，这是首选的"价格走势"数据源。

② 千岛 APP（原葩趣）—— 最垂直、品类最全

千岛（前身"葩趣"）是泡泡玛特投资的二手潮玩交易平台，专门做潮玩，品类覆盖最全，含大量官方系外的玩家交易。媒体也常引"千岛 APP 显示/成交均价"。如果你要跟踪某个冷门 IP、某个限定款的细颗粒价格，千岛比得物更全。它是泡泡玛特生态内的"官方系二手"，理论上最贴近真实成交。

③ 闲鱼 —— 量大但最乱，适合看"量"和"破发"信号

闲鱼成交量最大，但个人卖家多、挂牌价 ≠ 成交价，价格体系最不透明。它的价值不在"精确价格"，而在两个信号：① 挂牌量（放量 = 抛压加重）；② "破发"广度（多少款跌破了官方发售价）。适合做"广度扫描"，不适合做"精确走势"。

怎么看才准（四个要点）

  • 看"成交均价"，不是"挂牌价"：黄牛常挂天价引流，挂牌价严重失真；成交均价（得物/千岛都提供）才是真实博弈结果。
  • 看趋势，不是单点：某一天的价格受偶发交易影响很大。要取连续 4–8 周的移动趋势，才能判断是"波动"还是"趋势性折价扩大"。
  • 重点盯三类信号：隐藏款成交价（情绪锚）、整盒/端盒价（黄牛库存水位）、破发比例（跌破官方价的款数占比）——破发比例上升是最硬的"信仰松动"证据。
  • 跨平台交叉验证：得物 + 千岛 + 闲鱼三方对照，避免单一平台样本偏差（如某平台活动导致短期异动）。

三个必须知道的数据陷阱

  • 黄牛挂高价 ≠ 有人在买：很多"天价"是引流或试探，别被峰值吓到也别被峰值误导。
  • "补货即降价"是规律，不是例外：泡泡玛特持续扩产能、大规模补货，是二手价降温的关键推手。看到官方补货公告，二手价下行是确定性事件。
  • 未发货阶段的溢价是"纸面财富"：如 Labubu 冰箱开售 3 小时从 20999 元跌到 7300 元——那是未发货时"稀缺想象"的泡沫，实物一发货、套利盘集中抛售就崩。别把发售瞬间的峰值当真实需求。
补充：除 APP 外，财联社、新浪财经、界面新闻会做季度性的潮玩二手价盘点报道（如"Labubu 二手价全线下跌"），适合做低频的"大势确认"。雪球/公众号也有个人整理的"Labubu 价格跟踪"系列，可关注但需自行甄别样本口径。

五、收口：回到"130亿"那个命题

段永平赌的是"130亿是去峰值后的常态化底"，你的疑虑是"130亿是 Labubu 峰值"。这俩分歧，最终只能靠折价率（+单店坪效+新IP动销）来收敛——因为财报会滞后一年才告诉你答案，而那时候股价早就反映了。折价率处在所有指标最"前瞻"的位置，且逼问的是利润表永远回答不了的那个"信仰"问题。

一个值得记下的实时注脚：就在本文整理的前一天（2026-07-11），财联社报道段永平再度增持泡泡玛特，同日德银研报基于"Labubu 热度降温 + 海外表现较弱"下调预期、预计 Q2 增速放缓。多空两方的动作，恰好都落在"折价率接下来怎么走"这一个变量上。谁更接近真相，不在今天的口水里，而在二手市场接下来的走势里。

一句话：折价率是"烟雾报警器"——响了不一定着火，但不响你睡得着；持续响就得去查。利润是"年度体检报告"——最权威，但查出病时往往已晚了半年。新 IP 成功率是"基因测序"——决定长期寿命，但一年才出一次结果。三者共振，才构成对泡泡玛特完整的判断。
