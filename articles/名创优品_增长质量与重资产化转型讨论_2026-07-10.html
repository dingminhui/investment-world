名创优品讨论记录：增长质量、重资产化转型与观察仓决策
本文为 2026-07-10 与用户的讨论沉淀，基于当日「周五名创优品+泡泡玛特自动化」文章（含国元证券研报、GrowthInvesting 完整档案、经济观察网成本分析）及用户两轮追问整理。仅记录观点与验证框架，非买卖建议。

一、核心诊断：增量净利率仅约 3.8%，增长是"买来的"而非"赚来的"

以 2026Q1 数据（国元证券：营收 56.88 亿、+28.5%；经调整净利约 6.33 亿、+8.1%）倒推：

  • 去年同期营收 ≈ 44.26 亿，经调整净利 ≈ 5.855 亿，净利率 ≈ 13.2%
  • 今年 Q1 净利率 ≈ 11.1%
  • 新增收入 12.62 亿，新增净利仅 0.475 亿 → 增量净利率 ≈ 3.8%

即每多赚 1 元收入，只有约 3.8 分落到经调整净利，其余 96 分被成本吞噬。这与"有护城河公司增量净利率应高于存量净利率（规模效应释放）"相反，呈现典型的"用钱买增长"特征。

二、资产端风险（用户首要关注）：欧美直营 + 大店 capex 吞噬现金流

2.1 事实纠正：海外并非"大多自营"，但新增直营全砸在最贵的市场

时点海外直营占比说明
2024 年中12.5%去年同期
2025 Q217.5%—
2026 Q1约 19.5%–20.6%截至 3/31，海外共 3,617 家，直营约 720–745 家
  • 存量上：海外约 80% 仍是轻资产合伙人 / 代理 / 经销模式——这是名创能在 90 国快速铺开、又不撑爆资产负债表的关键。
  • 方向上：直营占比一年从 12.5% 提到 20%，且新增直营几乎全在北美、欧洲等高购买力市场（北美门店 234→394 家）。

含义：欧美是租金、人力、爬坡最慢的市场，直营化的单店资本开支强度是东南亚合伙人店的数倍。即便直营只占 20%，其吃掉的 capex 与拉低的短期利润率远超该比例给人的直觉——这正是"销售分销费 +44% 且刚性"的根源（关店才能省，不像投放可随时停）。

2.2 腾笼换鸟：把名创从"轻"推向"重"

  • 过渡期双重失血：老店关 / 改造期间现金流中断，新大店装修投入 + 爬坡期不盈利，中间"只出不进"。
  • 模式质变：MINISO LAND 旗舰大店（时代广场、香榭丽舍）店效约普通店 9 倍，但单店 capex 与租金数量级更高，且高度依赖顶级商圈——本质是从"杂货零售"向"体验式 IP 零售"升维，赌品牌溢价而非极致性价比。

用户判断：门店升级、扩大店面、改善装潢档次方向都对，但增加了太多资本性投入。这与"欧美直营"叠加，构成最主要的资产端压力——FCF 转化率与 ROIC 在 capex 高峰期内承压。

三、IP 端风险（次关注）：自有 IP 慢是生意本质，拐点未到前是成本项

  • 用户指出：自名创宣布转向自有 IP 战略以来，实质进展慢，自有 IP 仍很少、占比很低。此判断与数据一致（授权费 +42% 增速 >> 自有 IP 占比提升速度）。
  • 慢有结构性原因：IP 孵化是"命中率"生意，不是"投入产出"生意（泡泡玛特 Molly 养近十年、Labubu 熬 6 年）；而名创核心能力是供应链 + 零售效率 + 711 快上新（广撒网、快迭代），与"深挖一个形象、长期养人设"的内容逻辑相反。让零售效率公司做内容孵化，等于短跑运动员跑马拉松。
  • 现状=重租轻建：IP 授权费飙升是向迪士尼 / 三丽鸥 / 环球交的"过路费"；TOP TOY（+51.4%）是唯一像样的自有平台苗子，但同样大量依赖外部 IP 分销。拐点（自有 IP 销售占比真正爬起）未到前，IP 战略是成本项而非资产项。

四、用户观点与结论

① 风险排序：用户更关心"资产端风险"（欧美直营 + 大店 capex 吞噬现金流 → 利润率回不去），而非 IP 端风险。

② 对重资产化的定性：主动重资产化体现了管理层主动拥抱变化的魄力和勇气；若能成功，名创优品将到达新一个层级（对标"无印良品 + 泡泡玛特"混合体）。

③ 投资决策：总体看，名创优品当前悲观股价值得关注，并在低点买入观察仓。

逻辑链条：转型方向性感但必然经历利润率下行、capex 上升、FCF 转化变差（即 Q1 数据）；成败不取决于开多少店，而取决于两个慢变量——① 自有 IP 占比能否真正爬起；② 欧美直营大店的同店销售与店效能否证明品牌溢价成立。观察仓的意义：用有限仓位参与"若成功则跃迁"的左侧期权，同时避免重仓押注转型必成。
